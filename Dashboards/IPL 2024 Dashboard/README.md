## IPL Live Scoreboard and Points Table Dashboard
This Power BI dashboard provides a comprehensive and visually appealing way to track the latest scores and points table for the ongoing Indian Premier League (IPL) season. The dashboard fetches live data from an IPL API, ensuring that the information displayed is always up-to-date.

### Features:
1. *Live Scoreboard*
   - Displays the current score, overs, and other essential details for the ongoing IPL match.
   - Updates in real-time as the match progresses, providing users with the latest scoring updates.
   - Includes team logos, batsman and bowler details, and other relevant information.
2. *Points Table*
   - Presents the latest IPL points table, reflecting the current standings of all teams.
   - Showcases essential statistics such as matches played, wins, losses, net run rate, and points for each team.
   - Sorted based on the points tally, making it easy to identify the top-performing teams.
3. *Visually Appealing Design*
   - Utilizes a modern and clean layout, ensuring optimal readability and user experience.
   - Incorporates team logos, enhancing the visual appeal and brand recognition.
   - Responsive design adapts to different screen sizes and devices.
4. *Data Refreshing*
   - Manual refresh option available for users to fetch the latest data on demand.
5. *Filtering and Sorting*
   - Allows users to filter the data based on various criteria, such as team names or match dates.
   - Sorting options enable users to reorder the points table based on different statistics.
